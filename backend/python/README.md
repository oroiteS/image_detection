# Python Backend

电力巡检图像智能检测系统的Python后端部分。

## 目录结构

```
backend/python/
├── src/
│   └── image_detection/
│       ├── core/         # 核心检测模块
│       ├── data/         # 数据处理模块
│       ├── training/     # 模型训练模块
│       ├── utils/        # 工具函数
│       └── web/          # Web接口
├── tests/                # 单元测试
└── pyproject.toml        # 项目配置
```

## 安装

```bash
cd backend/python
uv sync
```
