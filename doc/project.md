# 电力巡检图像智能检测与分析系统

## 项目目标
围绕电力巡检图像缺陷检测任务，构建“数据管理与标注 → 训练评估 → 推理加速 → 服务部署 → 结果展示”的完整闭环系统，覆盖四类缺陷：绝缘子缺失、绝缘子烧蚀、鸟巢、均压环移位。

## 基础项目结构（现有）
- `backend/python/`：Python 后端，包含数据处理、训练与评估、Web 接口能力（PyTorch/Ultralytics YOLO/FastAPI/Gradio）。
- `backend/cpp/`：C++ 后端，面向高性能推理与部署（C++17/CMake/OpenCV）。
- `frontend/`：前端界面工程（Vue 3 + Vite），用于业务管理与结果展示的可视化页面。

## 功能需求拆解
### 1. 巡检数据管理与标注
- 巡检图像数据导入：支持手动上传与批量导入。
- 缺陷标注：集成 labelImg，支持 YOLO 与 PASCAL VOC 两种格式，提供标注版本管理。
- 数据查询：统计缺陷数量、分布与标注版本信息。

### 2. 智能检测模型训练与评估
- 采用 YOLOv11 进行端到端目标检测训练。
- 数据预处理与增强：尺寸统一、随机翻转/旋转/色彩调整；训练/验证/测试划分。
- 训练可视化：实时展示 loss、precision、recall、mAP 等指标。
- 评估报告：混淆矩阵、PR 曲线、检测可视化示例。

### 3. 模型优化与加速
- 转换链路：PyTorch → ONNX → TensorRT 引擎。
- 图优化：层融合、常量折叠、冗余计算消除。
- 精度优化：FP32/FP16/INT8 混合精度；INT8 量化含校准流程。
- 内核自动调优与后处理 CUDA 融合（边界框解码等）。
- 多批次并行推理；pybind11 封装 C++ 推理模块供 Python 调用。

### 4. 推理服务与结果展示
- 高性能推理引擎：TensorRT C++ API，支持动态输入尺寸与动态批次。
- RESTful API：JSON 请求/响应，支持单图与批量检测。
- 并发处理：多线程并发，保证高并发稳定性。
- 结果可视化：检测框叠加、类别与置信度展示、检测详情列表。
- 进阶功能：实时视频流检测（加分项）。

### 5. 模型效果分析
- 不同版本模型的对比分析。
- 误检/漏检样本分析与改进建议。

## 技术约束（必须满足）
- 标注工具：labelImg；输出 YOLO 与 PASCAL VOC 格式。
- 训练框架：PyTorch 1.12+，CUDA 11.x+。
- 模型转换：PyTorch → ONNX。
- 推理加速：TensorRT 8.x+；图优化与 INT8 量化。
- 服务语言：C++17；OpenCV 4.x 读写与展示；CUDA 后处理融合。
- Web 界面：Gradio（模型演示）+ 前端页面（管理与展示）。
- API：RESTful + JSON。
